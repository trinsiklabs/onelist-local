Asset Enrichment Agent UI Plan

Document Version: 2026-01-30 Status: Active Parent: ui_plan.md Related: shared_agent_ui_components.md, settings_ui_plan.md, reader_ui_plan.md

1. Overview

This document defines UI components specific to the Asset Enrichment Agent. For shared agent UI patterns (processing status, activity items, cost display), see shared_agent_ui_components.md.

Asset Enrichment Responsibilities:
  • Audio/video transcription (Whisper)
  • Image description and OCR
  • Action item extraction from transcripts
  • Decision extraction from transcripts
  • Tag suggestions based on content
    • Enrichment Types:
  • Transcript: Full text with timed segments
  • Description: AI-generated image/video description
  • OCR: Extracted text from images/PDFs
  • Extracted Items: Action items, decisions (become linked entries)

    • 2. Transcript Viewer

    Displays transcription results for audio/video assets with speaker diarization, read-along highlighting, and inline editing.

    2.1 Full Transcript View

    Located in entry detail under the asset or in a dedicated tab. Displayed as a data table with timestamp, speaker, and transcript columns. Supports speaker identification, word-level highlighting during playback, and user editing.
