Document Version: 2026-01-30 Status: Active Parent: ui_plan.md Related: shared_agent_ui_components.md, settings_ui_plan.md

1. Overview

This document defines UI components specific to the Searcher Agent. For shared agent UI patterns (processing status, activity items, duplicate detection), see shared_agent_ui_components.md.

Searcher Responsibilities:
  • Embedding generation and management
  • Semantic search (vector similarity)
  • Keyword search (full-text)
  • Hybrid search (combined)
  • Pre-storage similarity checking
  • Similar entries discovery

    • 2. Search Results Display

    2.1 Score Display Philosophy

  • Default: Simple match percentage only
  • Hover/Expanded: Full breakdown (semantic + keyword scores)
